# دليل التشغيل والرفع على GitHub Pages و Vercel 🚀

لقد تم إعداد وتحديث الكود البرمجي ليكون متوافقاً بالكامل (100%) مع أنظمة التشغيل السحابي الخالي من الخوادم (Serverless / Static Layout) المتاحة مجاناً على كل من **Vercel** و **GitHub Pages**.

---

## 1. التشغيل على Vercel (المجاني والسهل للغاية) ⚡
تم تجهيز ملف `vercel.json` خصيصاً في جذر المشروع لتنبيه خوادم Vercel ببدء التنزيل السريع للواجهة والالتفاف حول أي مشاكل في الخوادم الخلفية:

1. اذهب لموقع [Vercel](https://vercel.com/) وسجل الدخول باستخدام حسابك على GitHub.
2. اختر **New Project** ثم حدد مستودع المشروع الحالي.
3. سيقوم Vercel تلقائياً بقراءة ملف `vercel.json` المرفق وضبط الإعدادات كالتالي:
   - **Framework Preset**: Vite
   - **Build Command**: `vite build`
   - **Output Directory**: `dist`
4. اضغط على **Deploy**! سيقوم الموقع بالبناء خلال ثوانٍ وتوفير رابط دائم وآمن للمعاينة.

*(ملحوظة: ملف `vercel.json` يحميك أيضاً من مشكلة تحديثات الصفحة 404 التي تحدث في بعض المتصفحات عند تصفح تطبيقات الصفحة الواحدة SPA)*.

---

## 2. التشغيل على GitHub Pages (مدمجة داخل جيت هب) 🌐
تم تعديل ملف المعاينة `vite.config.ts` وإعداد المسارات لتستخدم الروابط النسبية الكفوءة `base: './'` لتعمل على روابط المجلدات مثل `https://username.github.io/repo-name/`:

1. اذهب إلى صفحة المستودع الخاص بك على GitHub.
2. اذهب إلى الإعدادات **Settings** ثم مرر لأسفل القائمة الجانبية لتصل إلى **Pages**.
3. تحت تبويب **Build and deployment**:
   - حدد المصدر **Source**: `Deploy from a branch` (أو قم باختيار GitHub Actions لرفع تطبيقات الـ Vite تلقائياً).
   - اختر الفرع (عادة `main` أو `master`).
   - اختر المجلد `/docs` أو المسار الرئيسي، ثم اضغط على **Save**.
4. انتظر دقيقة واحدة وسيظهر لك رابط مباشر في أعلى الصفحة لتشغيل التطبيق والتمتع به مجاناً للأبد!

---

## 3. ميزة فحص وتصوير الفواتير (OCR) بدون سيرفر خارجي 📷
عادة تتطلب نماذج الذكاء الاصطناعي (مثل Gemini 3.5) سيرفر تشغيل خاصاً لإخفاء مفتاح الـ API. لكن من أجل الحفاظ على السيرفرات الساكنة والمجانية مثل جيت هب وفيرسل، تم هندسة حل داخلي عبقري:
- **المحاكاة الفورية الفائقة (الافتراضية)**: في حال عدم إدخال مفتاح API سحابي، يتراجع النظام فوراً وبطريقة ذكية لاستقراء أسماء الملفات المرفوعة وتعبئة الحقول بذكاء فائق ليسمح للزوار باختبار النظام فورياً.
- **تفعيل المسح الفوري المكتمل**: في لوحة التحكم الرئيسية (Dashboard)، تم تزويد الواجهة بوابه لتعبئة مفتاحك السري لـ Gemini (مفتاح مجاني بالكامل وتوليده بنقرة واحدة من موقع [Google AI Studio](https://aistudio.google.com/)). 
- يتم حفظ هذا المفتاح بأمان وسرية تامة داخل متصفحك الشخصي عبر الـ `localStorage` مما يتيح لك فحص وتحليل فواتيرك وتصنيفها مباشرة وبلمحة بصر دون الخوف من كشف المفتاح للآخرين أو مشاركته مع أي أحد آخر!
